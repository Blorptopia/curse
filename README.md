# Curse
Curse is our little gamejam game for [Juniper's Very Serious Gamejam](https://itch.io/jam/theveryseriousjuniperdevgamejam). 

The game revolves around brewing potions and serving it to your customers.

You can [play the game yourself](https://curse.blorptopia.workers.dev), or you can look at [our gamejam page on itch](https://itch.io/jam/theveryseriousjuniperdevgamejam/rate/4715836) if you would like to see what others think instead.
